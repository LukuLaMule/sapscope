Design a login page for a SaaS B2B web app called SAPscope (SAP monitoring tool for IT consultants).

The page contains a single centered card with:
- App name and tagline "SAP Landscape Intelligence"
- Email field
- Password field
- Submit button "Sign in"
- Error message area (shown on bad credentials)
- Link "Forgot password?" → shows a new form in the same card (email field + send button + confirmation message)
- Link "Create account" → shows a register form in the same card (email, password, confirm password, submit)

No sidebar, no navigation. Just the card centered on the page.
