# Contributing to SAPscope

Thanks for your interest in SAPscope.

## Bug reports

Open a [GitHub Issue](https://github.com/LukuLaMule/sapscope/issues/new?template=bug_report.md) with:
- SAPscope version / Docker image tag
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs (`docker compose logs backend`)

## Feature requests

Open a [GitHub Issue](https://github.com/LukuLaMule/sapscope/issues/new?template=feature_request.md) describing:
- The SAP Basis use case / pain point
- What you'd expect SAPscope to do

## Pull requests

1. Open an issue first to discuss the change
2. Fork and create a feature branch
3. Run the test suite: `./run-tests.sh`
4. Submit a PR — a maintainer will review it
